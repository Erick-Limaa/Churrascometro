Churrascômetro

Política de Privacidade

✅ Sem coleta de dados pessoais

📋 Visão Geral

O Churrascômetro é um aplicativo de organização de churrascos desenvolvido para Android. Esta política descreve como o aplicativo lida com as informações inseridas pelos usuários.

Sua privacidade é importante para nós. O Churrascômetro foi desenvolvido com o princípio de coletar o mínimo possível de dados e nunca compartilhá-los com terceiros.

💾 Dados Armazenados

O aplicativo armazena localmente no seu dispositivo as seguintes informações, exclusivamente para o funcionamento do app:

• Nomes dos eventos criados pelo usuário
• Nomes dos convidados adicionados
• Configurações do cardápio (itens, preços e quantidades)
• Histórico dos churras calculados
• Estado do checklist de compras

Todos esses dados ficam armazenados apenas no seu dispositivo, utilizando o AsyncStorage do React Native. Nenhuma informação é enviada para servidores externos ou terceiros.

🚫 O Que Não Coletamos

O Churrascômetro não coleta:

• Dados pessoais identificáveis (nome, e-mail, CPF, etc.)
• Localização geográfica
• Contatos do dispositivo
• Dados de uso ou analytics
• Informações financeiras
• Cookies ou rastreadores

🌐 Conexão com Internet

O aplicativo pode solicitar acesso à internet para a funcionalidade de compartilhamento via WhatsApp e outros aplicativos de mensagem. Essa funcionalidade utiliza apenas a API nativa de compartilhamento do Android e não envia dados para servidores do Churrascômetro.

🗑️ Exclusão de Dados

Todos os dados armazenados pelo aplicativo podem ser excluídos a qualquer momento:

• Histórico: use o botão "Limpar histórico" dentro do app
• Todos os dados: desinstale o aplicativo do seu dispositivo

A desinstalação do aplicativo remove completamente todos os dados armazenados localmente.

👶 Crianças

O Churrascômetro não é direcionado a crianças menores de 13 anos e não coleta intencionalmente informações de menores de idade.

🔄 Alterações nesta Política

Esta política pode ser atualizada periodicamente. Qualquer alteração significativa será comunicada por meio de atualização do aplicativo na Google Play Store. A data da última atualização está indicada no rodapé desta página.
